# 🚀 Frontend Developer Portfolio

Welcome to my personal portfolio repository! This website serves as a digital home to showcase my skills, technical capabilities, and journey as a frontend developer. It features a fully responsive design, dynamic micro-interactions, interactive text effects, and a global theme-switching system.

---

## 🛠️ Tech Stack & Ecosystem

This portfolio is engineered using modern frontend tools and libraries to ensure fast rendering, modular architecture, and an engaging user experience:

* **React:** Core framework utilized for building modular, reusable functional components.
* **React Router:** Handles seamless, client-side single-page application (SPA) navigation.
* **Context API:** Powers global state management, specifically driving the application's real-time light and dark theme engine.
* **AOS (Animate on Scroll):** Manages scroll-triggered animations to reveal layout elements dynamically as the user explores the page.
* **Typed.js:** Powers interactive, dynamic typing animations on the hero sections.
* **Particles.js:** Integrates smooth, responsive background particle networks for an immersive visual layer.
* **React CountUp:** Creates asynchronous, animated numerical counters for displaying milestones.

---

## ✨ Key Technical Features

* **Global Theme Management:** Context-driven state toggling that instantly updates UI tokens across all layout components.
* **Scroll-Driven Reveal Effects:** AOS configuration integrated across major layout sections to orchestrate natural element entrances based on viewport positions.
* **Fluid Layouts:** Clean structural design prioritizing high scannability and intuitive interface navigation.
* **Micro-Interactions:** Custom CSS transitions and scaling effects on interactive elements like navigation links without inducing layout shift.
* **Performance Optimization:** Built using optimized component lifecycles to ensure animations run at a smooth frame rate.

## 🚀 Local Installation & Setup

Follow these steps to clone the repository and spin up the development server on your machine:

1. **Clone the Repository:**
```bash
git clone https://github.com/MasterTechCode/portfolio.git

```


2. **Navigate into the Directory:**
```bash
cd portfolio

```


3. **Install Dependencies:**
```bash
npm install

```


4. **Launch the Local Server:**
```bash
npm run dev

```


Once running, open the local address (typically `http://localhost:5173`) in your web browser.
